自我认知（六）：我们的自尊

发表于 2017-11-27 | 分类于 自我认知
这篇博文是《恰如其分的自尊》[1]和《自我》[2]这两本书的整理和总结，核心内容是自尊对我们的影响和如何提高自尊水平。自尊描述的是我们如何看待、评估自己的价值，它是前面博文中所讲过的价值观、人格、情绪的基础，同时它也很大程度上受到原生家庭和学校教育的影响，在它基本成形之后，就在我们的鼻梁上悄悄的戴上了一副彩色的眼镜，我们所看到的事物都或多或少的受到了这副眼镜的影响。我们所处的社会和时代崇尚的永远是高自尊者，在媒体上频频露面的企业家、政客、学者无不表现出高自尊所具有的特征，相比之下，低自尊的人所受的关注和获得的资源也少得多，所以，尽管高自尊的人也有其缺点所在，但是低自尊的人所承受的痛苦和压力会更大，《恰如其分的自尊》这本书的主要目的也是帮助那些自尊水平较低的人，可以配合另外两本专门讲述自信重建的书《医治受伤的自信》[4]和《世界偏爱自愈自乐的你》[6]一起来看。

自我认知（五）：焦虑分子

发表于 2017-11-14 | 分类于 自我认知
焦虑是我们这个时代的底色，大家为自己的未来焦虑，为没有男女朋友焦虑，为买不起房而焦虑，这几年刚出现的知识焦虑[2]，以及网络上年轻人称自己是需要靠枸杞养生的老年人这一现象背后所体现的对自己健康的焦虑。群体的情绪会下意识的影响到群体中的个体，焦虑不可避免，但是过分焦虑是不必要的。三联书店知心书系列[3]中的这本《你好，焦虑分子》[1]集中关注焦虑，详细的分析了焦虑产生的原因、焦虑本身的作用、几种主要的焦虑类型以及如何调节不正常的焦虑情绪。

自我认知（四）：俘虏完美主义

发表于 2017-11-01 | 分类于 自我认知
标题中的“俘虏”做了动词来用，之所以取这样一个有点怪的名字是因为控制权的问题：我无法彻底消除自身的完美主义倾向，但是我可以控制它。我以前是完美主义者，也深受完美主义的苦，但是苦于找不到问题出在哪里。就像很多人面对黑暗会心生恐惧，我面对不完美也是这种感觉，这种感觉下意识的控制着我的行为，让我深陷于各种细节之中而无法看清大局：一本书必须从头到尾看完才能放心、一部美剧必须以每一季为单位观看、任何一件小事没有做好都会让我自责不已.......我意识到问题的严重性之后，一直在寻找解脱的办法，最终，这本只有180页的《从自我苛求中解放出来》让我找到了问题的根源并最终让我具备控制自身完美主义倾向的能力。另外，三联书店的知心书系列又更新了，专门有一本讲强迫症的心理学著作《走出强迫症》。

自我认知（三）：逻辑谬误

发表于 2017-10-20 | 分类于 自我认知
情绪会影响我们的决策过程，除了做好情绪管理之外，是否还有其他办法来优化我们的决策呢？逻辑可以做到。大家经常听到的批判性思维，它要达到的目标就是“对于信什么，做什么，有合理的决定”[2]，这里面的“合理的决定”便是逻辑大展拳脚的地方。《好好讲道理-反击谬误的逻辑学训练》整本书语言干净利落，没有口水类废话，是一本干货纯度100%的书，这本书的核心内容是日常生活中常出现的60种逻辑谬误，被作者分为五大类。对于每一种谬误，作者给出了谬误的定义、谬误的问题所在、实例以及如何回击谬误四个部分，我选择了我自己平时容易犯的谬误进行了整理。墙裂建议大家去看原书，虽然在日常对话中不需要如此严肃的逻辑推理，但是在生活中许多需要做决策和选择的时候，拥有良好的逻辑推理能够帮助我们做出更合理的选择。这本书唯一可能的缺点就是语言偏严肃，有许多辩论上的专业名词，而另外一本同样著名的书《清醒思考的艺术》[6]则是用偏向生活化的语言来告诉我们如何做到清醒的思考。

自我认知（二）：基本情绪和情绪管理

发表于 2017-10-01 | 分类于 自我认知
《与生俱来的七情》这本书讲的是我们的基本情绪。如果把我们个体看作一个小黑盒，这个黑盒的输入端就是我们的信息接收器官，包括眼睛、耳朵、皮肤、鼻子等，这个黑盒的输出端就是我们的行为，包括表情、眼神、语速、音高、肢体动作等，这个黑盒的功能简单来说就是根据输入信息来调整输出信息，谁来调呢？怎么调呢？《思考，快与慢》的作者认为，这个黑盒包括两个部分：系统1和系统2，系统1根据输入信息产生情绪、直觉、意向等初步决策，系统2根据系统1产生的初步决策利用逻辑推理、怀疑和反思、自我控制等理性思维做出最终决策，即输出我们的行为。情绪的重要性体现在：它会影响系统2的理性思维过程。因此，了解基本情绪以及管理好自己的情绪的重要性也就不言而喻。
